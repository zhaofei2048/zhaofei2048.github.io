Tips on Python GIS

Updated: 周日 28 一月 2024
By Fei Zhao

In GIS.

tags: GIS Python

Nice Python packages for GIS:

  • Basics: pyproj (for projection); gdal (for raster data processing); fiona (for vector data io)
  • Geometry operation: shapely
  • Vector data: geopandas
  • Raster data: rasterio
  • Raster and vector data analysis: rasterstats
  • Map visualization: cartopy + geopandas + geoplot + matplotlib-scalebar + geo_northarrow
